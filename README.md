## Main Project: rvkit

> Bare metal Zig, without the bare metal pain.

I am the creator and maintainer of **rvkit**, a CLI/TUI toolchain written in Rust 
for Zig developers targeting RISC-V microcontrollers (like CH32V003 and ESP32-C3).

It fixes the fragmented toolchain pain by providing an all-in-one experience:
* [Scaffold] One command to generate a ready-to-build project with linker scripts.
* [Build]    Seamless integration with the native Zig build system.
* [Flash]    Automated firmware flashing via wlink or esptool.
* [Monitor]  Built-in TUI serial monitor powered by Ratatui.

-> Repository: [github.com/Karagure/rvkit](https://github.com/Karagure/rvkit) (or your repo link)
