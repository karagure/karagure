## 🛠 Main project

### [rvkit](https://github.com/karagure/rvkit) — *Bare metal Zig, without the bare metal pain*

A CLI/TUI toolchain in Rust for Zig developers targeting RISC-V microcontrollers 
(CH32V003, ESP32-C3). Handles project scaffolding, builds, flashing, and serial 
monitoring in one tool.

- **Stack**: Rust, Zig, Ratatui
- **Status**: v0.1.1 alpha · [pre-built binaries](https://github.com/karagure/rvkit/releases/latest)
- **Why**: Setting up bare-metal Zig on RISC-V should not take a weekend.
